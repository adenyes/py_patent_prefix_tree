# py_patent_prefix_tree
Parse highly repetitive list of text strings in an XML document from (e.g. from government documents you are trying to skim) into a tree structure.

Optionally take an --xpath to show where to pick up the text.

Example:
```
prefix_tree.py input/APP0001.html --xpath "trademark-case-files/trademark-case-file/base-application-form/goods-services/goods-service[1]/description-text"
```
